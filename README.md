# Zephyr-Zulip Bridge

[Zulip](https://zulip.com) is an open-source team collaboration tool bearing
some similarity to [Zephyr](http://zephyr-im.org/).  This repo is a fork of
Zulip with additional changes for our instance of Zulip and its use as a shared
Zephyr client.

Zulip is distributed under the [Apache
2.0](https://github.com/zulip/zulip/blob/main/LICENSE) license.
